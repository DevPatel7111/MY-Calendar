01 — Setup

Enter your Google OAuth 2.0 Client ID to connect this app directly to your Google Calendar. Your events are sent straight from your browser to Google — no server involved.

Don't have a Client ID? Follow these steps:
  1. Open console.cloud.google.com and create a project.
  2. Go to APIs & Services → Library and enable Google Calendar API.
  3. Go to APIs & Services → OAuth consent screen → External → fill app name → Save.
  4. Go to Credentials → Create Credentials → OAuth Client ID → choose Web application.
  5. Under Authorized JavaScript origins add: https://devpatel7111.github.io
  6. Copy the Client ID and paste it below.
